# Arquivos de Configuração
Espaço reservado para armazenar backups das regras do pfSense (XML), configurações do Agente Wazuh e certificados da VPN.
# Configurações do Ambiente

Nesta pasta, encontrará os parâmetros necessários para interligar as peças do laboratório:

1. **Firewall Rules**: Lista de portas que precisam de ser abertas no pfSense.
2. **Wazuh Agent**: Modelo de como configurar os endpoints para comunicarem com o servidor central.

> **Dica:** Após configurar o seu pfSense, pode exportar o ficheiro `config.xml` (removendo senhas e dados sensíveis) e colocá-lo aqui como backup.
